Guide to the pdf report

Once the module is installed, you will find in the "Settings" item "HexCode pdf report". Through this form you can create an unlimited number of custom themes for your report. "Click on create" to create a new theme. This opens the creation form (see next screenshot). In this form, you have two choices: Use a Report Template already created (by default there are 6) or create a custom Theme. In the first case it is sufficient to name the theme, and select from the dropdown Report Template one of the six ready-made templates. In the second case, it is obligatory to put the tick "Custom Report" and then enter the hexadecimal color codes in the various sections. Once created and finished the theme click on save.

Print your first Report

download a report in pdf has never been easier!

Go to one of the sections listed in the first paragraph, Example "Invoice", open an invoice in the form at the top you will find a menu "Print." If you click on it you will find the voice "HEXCode invoice pdf". Just click this to download your invoice !. The same procedure must be performed to download claims, estimates etc.

Change theme for each Form

thanks to the form "HEXCode pdf report" in the "settings" section, you can create a number of records associated with one of the six themes. These records must be associated in the view form of a complaint or an invoice etc .. to give the desired theme. E 'enough to go into the "Style PDF report" and select the name record.

Wkhtmltopdf necessary for operation

Follow these easy steps

Odoo to download the report in PDF format uses the best plugin to convert a page to PDF format. This plugin is called: wkhtmltopdf. If in your system where you install odoo you have not already installed it is necessary to do so. It 'just download the package from the official website and install it on the machine where it runs odoo.
